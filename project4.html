Water, Earth, Fire, and Data: The World of Avatar the Last Airbender

Overview

In my Tableau dashboard, “Water, Earth, Fire, Air, and Data,” I conducted an in-depth analysis of Avatar: The Last Airbender using IMDb data. This project was part of the 2025 Iron Viz: Student Edition competition, which challenged participants to create compelling visualizations centered around television shows. The dashboard explores various aspects of the series, including episode ratings, character development, and thematic elements. By leveraging Tableau's interactive features, I aimed to uncover patterns and insights that contribute to the show's enduring popularity. This visualization not only showcases my data storytelling skills but also reflects my passion for combining data analysis with creative design (I also drew the illustrations included in this dashboard!).
